# Gmail-Account-Creation

Problem Statement:
Some times it get difficult for a user to create or find a desired user account with certain prefix number in it. This is because the email account has been already taken. A user has requested you to create an accont with certain prefix in it.

Problem Solution:
Script to Automate the process

Libraries, Language & Browser Used:
- Selenium
- Google Chrome driver
- Python 3.8

Pre-requiste:
- Any Python Editor


Gmail Account Creation is easy when we get the account id that we like..
But what about when we don't get the account that we want?
It gets troublesome when we want to create a certain account with likely number in it and don't get it.
The code Written in this repository helps you fix the problem for you.
